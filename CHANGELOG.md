# Changelog

All notable changes to this project are documented in this file.

The project currently uses a simple manual changelog approach.

## [0.2.0] - 2026-04-27

### Added

- initial public repository setup
- English and Chinese README files
- MIT license
- contributing guide
- example batch config for thumbnail generation
- GitHub issue templates and pull request template
- security policy and code of conduct

### Included In This Version

- reusable Python CLI for image conversion, resizing, square crop, square padding, metadata inspection, and batch processing
- built-in profiles for avatar, social square, square pad, thumbnail, WEBP export, and Full HD wallpaper workflows
- config-driven batch execution and JSON manifest output
