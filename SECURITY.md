# Security Policy

## Supported Versions

This repository is currently maintained on the latest `main` branch only.

If you discover a security-sensitive issue, please avoid opening a public issue first.

## Reporting A Vulnerability

Please report security issues privately by contacting the repository owner through GitHub.

When possible, include:

- affected command or workflow
- reproduction steps
- impact assessment
- screenshots or logs if relevant

I will review the report, confirm whether it is security-relevant, and follow up with a fix or mitigation guidance as appropriate.

## Scope Notes

Likely security-relevant areas include:

- unsafe path handling
- unintended overwrite behavior
- malformed input handling
- dangerous command construction around ImageMagick execution

General bugs, feature requests, and usability issues should go to the normal GitHub issue tracker instead.
