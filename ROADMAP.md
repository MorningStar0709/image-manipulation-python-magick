# Roadmap

This roadmap reflects practical improvements that would make the project more stable, easier to adopt, and easier to maintain.

## Near Term

- improve README examples for Linux and macOS shell usage
- add more profile examples with before-and-after oriented documentation
- refine error messages for common path, overwrite, and environment failures
- add a small sample input set for demonstration and validation

## Mid Term

- add focused automated tests for argument validation and config merging
- provide optional example workflows for common publishing scenarios
- improve manifest output with clearer summaries for skipped files
- document best practices for large batch jobs and safer output layouts

## Longer Term

- evaluate packaging options beyond direct script usage
- consider optional parallel batch execution with clear safety limits
- expand profile presets for more real-world publishing and ecommerce tasks
- improve cross-platform installation guidance

## Out Of Scope For Now

- GUI-based manual image editing
- semantic image understanding or AI-driven retouching
- heavy dependency stacks that weaken portability
